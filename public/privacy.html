Privacy Policy

Last updated: July 2026

Note: This is a non-commercial proof-of-concept application used to test a WhatsApp-based loan-onboarding conversation flow. It is not a live financial product, and no real loan applications are processed, approved, or submitted to any lender through this system.

What this application does

This application connects to the WhatsApp Cloud API to simulate a loan-application onboarding conversation. When a user messages the associated WhatsApp test number, the application responds with automated prompts to collect sample information as part of a demonstration loan-application journey.

Information we collect

During a conversation, the following information may be collected and stored:

  • WhatsApp phone number (used as an identifier for the conversation)
  • Name, date of birth, employment type, and stated monthly income
  • PAN (Permanent Account Number) as entered by the user
  • Media identifiers for any images uploaded (e.g. ID or address proof photos) via WhatsApp
  • The state of the conversation (which step of the demo flow the user has reached)

How information is used

Information collected is used solely to operate the demo conversation flow (e.g. remembering earlier answers, generating a sample reference number, and sending status-update messages back to the user). It is not sold, shared with third parties, or used for advertising. It is not submitted to any credit bureau, lender, or bank as part of this proof of concept.

Data storage

Conversation data is stored in a Supabase-hosted database associated with this project, accessible only to the developer operating this proof of concept. Uploaded images are referenced by their WhatsApp media ID; the underlying image files are not permanently downloaded or stored by default in this version of the application.

Data retention and deletion

Data is retained only for as long as needed to test and demonstrate the conversation flow. You may request deletion of any data associated with your WhatsApp number by contacting the developer using the details below.

Third-party services

This application relies on Meta's WhatsApp Cloud API to send and receive messages, and Supabase to store conversation data. Each of these providers has its own privacy practices governing data transmitted through their platforms.
